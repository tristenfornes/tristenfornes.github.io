Basic HTML

This assignment focuses on creating a simple webpage using basic HTML elements. Explore the structure of HTML documents and practice incorporating headers, paragraphs, images, lists, and links.

Basic CSS

Assignment 2 introduces basic CSS styling to enhance the visual appeal of web pages. Learn to apply styles such as colors, fonts, and borders to create a more polished and visually appealing user interface.

Page Layout

In Assignment 3, you'll dive into page layout techniques using CSS Flexbox. Understand how to create responsive and flexible layouts for various screen sizes, ensuring a seamless user experience.
