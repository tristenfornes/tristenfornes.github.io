Web Development Class

Assignments

  • Assignment 1 - Basic HTML

    This assignment focuses on creating a simple webpage using basic HTML elements. Explore the structure of HTML documents and practice incorporating headers, paragraphs, images, lists, and links.

  • Assignment 2 - Basic CSS

    Learn the fundamentals of CSS in this assignment. Apply styles to HTML elements, modify fonts, colors, and layout properties to enhance the visual appeal of your web pages.

  • Assignment 3 - Page Layout

    Explore advanced page layout techniques using CSS Flexbox. Create a responsive design that adapts to different screen sizes, providing a seamless user experience.

  • Assignment 7 - JavaScript Buttons and More

    Learn JavaScript by creating interactive buttons and more in this assignment. Explore the power of JavaScript in enhancing user interactivity on the web.

  • Assignment 8 - JavaScript Events

    Explore JavaScript events and their usage in web development. Learn how to handle user interactions effectively.

  • Assignment 9 - Bounce Ball & Yoga Moves

    Explore JavaScript by creating a bouncing ball animation and interactive yoga moves display.

  • Assignment 10 - Advertisement Banner and Images

    This assignment involves creating an advertisement banner and displaying images with attributions.
