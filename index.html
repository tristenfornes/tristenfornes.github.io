Web Development Class

Assignments

  • Assignment 1 - Basic HTML

    This assignment focuses on creating a simple webpage using basic HTML elements. Explore the structure of HTML documents and practice incorporating headers, paragraphs, images, lists, and links.

  • Assignment 2 - Basic CSS

    Learn the fundamentals of CSS in this assignment. Apply styles to HTML elements, modify fonts, colors, and layout properties to enhance the visual appeal of your web pages.

  • Assignment 3 - Page Layout

    Explore advanced page layout techniques using CSS Flexbox. Create a responsive design that adapts to different screen sizes, providing a seamless user experience.
